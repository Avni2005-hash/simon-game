# 🎮 Simon Says Game

A fun and interactive **Simon Says Memory Game** built using **HTML**, **CSS**, and **Vanilla JavaScript**.

Test your memory by repeating an increasingly longer sequence of colored buttons. Each level introduces a new color, making the challenge more exciting!

---

## ✨ Features

- 🎯 Interactive Simon Says gameplay
- 🎨 Random color sequence generation
- 📈 Level-based progression
- ⚡ Smooth button flash animations
- ✅ Real-time answer validation
- 🏆 Score displayed on game over
- 🔄 Restart the game with any key press
- 💻 Simple and responsive user interface

---

## 🛠️ Technologies Used

- 🌐 HTML5
- 🎨 CSS3
- ⚙️ JavaScript (ES6)

---

## 🎮 How to Play

1. ⌨️ Press **any key** to start the game.
2. 👀 Watch the highlighted color carefully.
3. 🖱️ Click the buttons in the **same order** as shown.
4. ➕ Every correct round adds a new color to the sequence.
5. 🧠 Keep remembering the growing pattern.
6. ❌ Click a wrong button and the game ends.
7. 🔁 Press any key to play again and try to beat your previous score!
